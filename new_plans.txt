1. Hacer una web no hace falta tan complejo, una web, que puedas configurar directamente, el bot

Ahora explico como poder confiurar el bot de un servidor en concreto
Al querer configurar un bot vas a necesitar un Token que la va a proporcionar el bot a la hora de llamar una funcion
por ejemplo /token este comando lo que hara es crear un token para el servidor actual
y a partir de eso, poder directamente introducir ese token en la web, para poder configurar el bot en el servidor actual
la configuracion, se guardara en una base de datos, en vez, de un fichero, para hacerlo mas seguro, y que no se elimine el fichero

esta parte esta clara, luego como almacenar la configuracion del bot en una base de datos y como poder acceder a esa informacion de manera sencilla

es posible creando una funcion que se conecte y lea la informacion dinamicamente en la base de datos, de manera que se pueda acceder a cualquier valor sin tener que escribir codigo extra
